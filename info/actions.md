## Add new action

import `Action.astro` from `components/rules/actions` in mdx file


### Example

```ts
import Action from './components/rules/actions/Action.astro' 

<Action traits={[['Trait name',2],['Trait name',3]]} action={1}>
Title
<div name="content">
Text
</div>
</Action>
```
