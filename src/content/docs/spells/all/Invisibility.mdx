Illusions bend light around the target, rendering it [invisible](/Conditions.aspx?ID=26). This makes it [undetected](/Conditions.aspx?ID=39) to all creatures, though the creatures can attempt to find the target, making it [hidden](/Conditions.aspx?ID=22) to them instead. If the target uses a [hostile action](/Rules.aspx?ID=300), the spell ends after that hostile action is completed.

---
**Heightened (4th)** The spell lasts 1 minute, but it doesn't end if the target uses a hostile action.
