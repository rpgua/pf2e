<Spell title={"Telekinetic Hand"} traits={[['Cantrip'], ['Concentrate'], ['Manipulate']]} action={2} pfs={"Standard"} level={1} url={"https://2e.aonprd.com/Spells.aspx?ID=1715"} legacy_name={"Mage Hand"} range={30} source={['Player Core pg. 362']} spell_type={"Cantrip"} target={"1 unattended object of Light bulk or less"} tradition={['Arcane', 'Occult']} type={"Spell"}>
You create a floating, magical hand, either [invisible](/Conditions.aspx?ID=26) or ghostlike, that grasps the target object and levitates it slowly up to 20 feet in any direction. When you Sustain the spell, you can move the object an additional 20 feet. If the object is in the air when the spell ends, the object falls.

---
**Heightened (3rd)** You can target an unattended object with a Bulk of 1 or less.

**Heightened (5th)** The range increases to 60 feet, and you can target an unattended object with a Bulk of 1 or less.

**Heightened (7th)** The range increases to 60 feet, and you can target an unattended object with a Bulk of 2 or less.

</Spell>
