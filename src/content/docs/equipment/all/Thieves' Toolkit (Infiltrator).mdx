 You need a thieves' toolkit to [Pick Locks](/Actions.aspx?ID=70) or [Disable Devices](/Actions.aspx?ID=69) (of some types) using the [Thievery](/Skills.aspx?ID=17) skill. If your thieves' toolkit is [broken](/Conditions.aspx?ID=2), you can repair it by replacing the lock picks with replacement picks appropriate to your toolkit; this doesn't require using the [Repair](/Actions.aspx?ID=42) action. If you wear your thieves' toolkit, you can draw and replace it as part of the action that uses it.
