 A repair toolkit allows you to perform simple repairs while traveling. It contains a portable anvil, tongs, woodworking tools, a whetstone, and oils for conditioning leather and wood. You can use a repair toolkit to [Repair](/Actions.aspx?ID=42) items using the [Crafting](/Skills.aspx?ID=4) skill. You can draw and replace a worn repair toolkit as part of the action that uses it.
