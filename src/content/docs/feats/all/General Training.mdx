  Your adaptability manifests in your mastery of a range of useful abilities. You gain a 1st-level [general feat](/Feats.aspx?Traits=78). You must meet the feat’s prerequisites, but if you select this feat during character creation, you can select the feat later in the process in order to determine which prerequisites you meet.

**Special** You can select this feat multiple times, choosing a different feat each time.
